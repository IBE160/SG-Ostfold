ibe160 – Application Color Theme

Mode: Dark-first

The color theme supports a “Modern & Tech-Forward” dark UI. It is optimized for dashboards and shift reports where data and anomalies must stand out without overloading the user’s eyes.

Modern & Tech-Forward (Dark)

A deep, low-noise background with a bright blue accent. Primary blue is reserved for the most important actions and key numbers; everything else uses more subtle contrasts.

Usage focus

Dashboard views, shift reports, historical data, and monitoring screens where users may look at the interface for long periods.

Primary contrast

Blue #3ABFF8 on dark surfaces for CTAs and highlights, white text for key numbers.

Accessibility

Avoid pure white blocks. Use content surfaces and secondary text for softer contrast.

Usage Guidelines

  • Use primary blue sparingly – ideally one main CTA per view.
  • Prefer secondary text for descriptions to reduce visual noise.
  • Keep backgrounds dark – avoid mixing light and dark modes in the same flow.
  • Semantic colors should not be overused; they should signal clear meaning (status, trend, alert).
  • Borders use border-border to subtly separate surfaces without hard lines.
